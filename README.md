## 👾 About The Project
> 🚧 **Project Status: Active Development (v0.1.0-alpha)**
> This API is currently in its initial development phase, but its core functionalities are already in place. Features are actively being added, and the database schema is subject to structural changes prior to the `1.0.0` release.

**Unstore API** is a backend service designed to manage service providers and retail businesses.
Inspired by real-world operational challenges I observed while working 
alongside my uncle, the project (whose name playfully nods to 'Uncle') aims to streamline store management, inventory, and service tracking.

### Key Features
* **Feature 1:** Store Management (Core Domain Modeling);
* **Feature 2:** Secure Auth via JWT;
* **Feature 3:** CRUD - WIP (Work In Progress)

---

## 🔮 Technologies Used

This project was built utilizing the following technologies, carefully chosen to balance productivity and performance:

| Technology | Purpose |
| :--- | :--- |
| **C# 14 / .NET 10** | Core language and framework. |
| **ASP.NET Core** | API routing, middleware, and request handling. |
| **Entity Framework Core** | Primary ORM for structural database migrations and complex domain modeling. |
| **SQLite** | Currently utilized for lightweight, zero-configuration development and testing. (Planned migration to SQL Server for production-grade relational data management). |
